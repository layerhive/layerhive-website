# LayerHive Launch-Ready Website

This is a clean, professional, responsive website for LayerHive custom 3D printing services.

## Files
- index.html
- styles.css
- script.js
- assets/layerhive-logo.jpeg

## How to launch for free
1. Go to https://vercel.com
2. Create a free account
3. Click "Add New Project"
4. Upload this folder or connect it through GitHub
5. Deploy
6. Add your domain later if needed

## Editing
- Text: edit `index.html`
- Colors/layout: edit `styles.css`
- Quote email logic: edit `script.js`
- Logo: replace `assets/layerhive-logo.jpeg`

## Important
The quote form opens the visitor's email app. For a real file upload form later, connect Formspree, Tally, Typeform, Netlify Forms, or a custom backend.
